# Chamba-De-Embebidos

## Descripción
Repositorio del proyecto de la materia Sistemas Embebidos.
Contiene prácticas y código desarrollado durante el curso.

## Objetivo
Aplicar conceptos básicos de sistemas embebidos mediante el uso de microcontroladores.

## Tecnologías
- Arduino
- Lenguaje C/C++
- IDE de Arduino
